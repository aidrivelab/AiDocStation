# 安全策略与隐私声明

### 漏洞反馈
如果您在本项目中发现潜在的安全漏洞，请**不要**直接公开发布 Issue。请通过以下途径与我们联系：
- 电子邮件：[code@pcfox.cn](mailto:code@pcfox.cn)（安全问题首选联系方式）
- GitHub Issues：用于反馈非安全类 Bug 和一般性建议。

### 隐私声明
AIDOC Station **Lite 版本**遵循隐私至上的设计原则：
- **离线处理**：所有文档转换逻辑均在您的本地计算机上完成。
- **无数据上传**：程序不会上传任何文档内容、用户隐私或个人敏感信息到云端。
- **本地存储**：所有配置信息与历史记录仅存储在您的本地应用程序目录中。

# Security Policy

### Reporting Vulnerabilities
If you discover a potential security vulnerability in this project, please **do not** open a public issue. Instead, report it through the following channels:
- Email: [code@pcfox.cn](mailto:code@pcfox.cn) (Primary contact for security issues)
- GitHub Issues: Use for non-security bugs and general feedback.

### Privacy Statement
The **Lite version** of AIDOC Station is designed with privacy as a priority:
- **Offline Processing**: All document conversions are handled locally on your machine.
- **No Data Upload**: No document content or personal information is uploaded to our servers.
- **Local Storage**: All configuration and history data are stored exclusively in your local application directory.
